Training Planner — это приложение для планирования и отслеживания тренировок с графическим интерфейсом.

### Описание приложения

**Training Planner** — это приложение для планирования и отслеживания тренировок с графическим интерфейсом. Программа позволяет:
* Добавлять новые тренировки
* Сохранять данные в JSON-формате
* Фильтровать тренировки по типу
* Проверять корректность вводимых данных

---

### Основные функции

* **Добавление тренировок:**
  * Ввод даты (формат ДД.ММ.ГГГГ)
  * Выбор типа тренировки
  * Указание длительности в минутах

* **Фильтрация:**
  * Поиск по типу тренировки
  * Отображение отфильтрованных результатов

* **Работа с данными:**
  * Автоматическое сохранение в JSON
  * Загрузка существующих данных

---

### Инструкция по использованию

1. **Запуск приложения:**
   * Откройте файл `training_planner.py`
   * Запустите программу

2. **Добавление тренировки:**
   * Заполните все поля формы
   * Нажмите кнопку "Добавить тренировку"

3. **Фильтрация:**
   * Введите тип тренировки в поле фильтра
   * Нажмите "Применить фильтр"

---

### Требования к вводу данных

* **Дата:** формат ДД.ММ.ГГГГ
* **Длительность:** положительное число
* **Тип тренировки:** произвольный текст

---

### Примеры использования

1. **Добавление новой тренировки:**
   * Дата: 01.05.2026
   * Тип: Кардио
   * Длительность: 45

2. **Фильтрация:**
   * Введите "Кардио" в поле фильтра
   * Нажмите "Применить фильтр"

---

### Возможные ошибки

* **Неверный формат даты:** появится сообщение об ошибке
* **Отрицательная длительность:** предупреждение о неверных данных
* **Пустые поля:** уведомление о необходимости заполнения

---

### Сохранение данных

* Все данные сохраняются в файл `trainings.json`
* При закрытии приложения данные автоматически сохраняются
* При запуске происходит загрузка сохраненных данных
